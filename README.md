# 🏦 Wipro Bank RD Account System

A simple Java application that manages a Recurring Deposit (RD) account using Object-Oriented Programming concepts.

## Features

- User input validation
- Custom exception handling
- Quarterly compound interest calculation
- Total deposited amount calculation
- Maturity amount calculation

## Technologies Used

- Java
- Eclipse IDE

## Project Structure

- `Account.java`
- `RDAccount.java`
- `BankService.java`
- `BankValidationException.java`
- `Main.java`

## Run the Project

1. Open the project in Eclipse.
2. Run `Main.java`.
3. Enter the required details.
4. View the interest, deposited amount, and maturity amount.
